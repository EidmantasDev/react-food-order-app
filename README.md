# React form

Food order app in React

## Run Locally

Clone the project

```bash
  git clone git@github.com:EidmantasDev/react-food-order-app.git
```

Go to the project directory

```bash
  cd react-food-order-app
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
